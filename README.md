# AgencyFlow 🚀

Plataforma de produtividade para agências de marketing digital + Orquestração multi-agente AI.

## Estrutura

```
/
├── app/              ← Web App (frontend)
│   ├── src/          ← Código fonte
│   ├── tests/        ← Testes unitários
│   └── supabase/     ← Schema SQL
├── _expxagents/      ← Engine de orquestração multi-agente
├── agents/           ← Definições de agentes AI (16 setores)
├── squads/           ← Squads configurados
├── mcps/             ← Integrações MCP (GitHub, Slack, etc.)
└── .github/          ← CI/CD
```

## Deploy Rápido (Vercel)

1. Crie conta em [vercel.com](https://vercel.com) (login com GitHub)
2. Suba este repositório para o GitHub
3. No Vercel: "Add New → Project" → importe o repositório
4. Root Directory: `app/`
5. Deploy automático em cada push 🚀
